二次函数的图像与性质
二次函数的一般形式为 y = ax² + bx + c（a ≠ 0），它的图像是一条抛物线，是高中数学中最重要内容之一。
当 a > 0 时，抛物线开口向上，最低点称为顶点；当 a < 0 时，开口向下，最高点为顶点。顶点坐标公式为 (-b/(2a), f(-b/(2a)))，或配方后写作 y = a(x - h)² + k，其中 (h, k) 即为顶点。
抛物线关于直线 x = -b/(2a) 对称，这条直线称为对称轴。与 y 轴交点为 (0, c)，称为 y 轴截距。与 x 轴的交点称为根，可通过求解 ax² + bx + c = 0 得到。若判别式 Δ = b² - 4ac > 0，有两个不同实根，抛物线与 x 轴交于两点；Δ = 0，有一个实根（顶点在 x 轴上）；Δ < 0，无实根，抛物线完全在 x 轴上方或下方。
二次函数的单调性也很清晰：在对称轴左侧，当 a > 0 时函数递减，a < 0 时递增；对称轴右侧则相反。这使得二次函数成为研究最值、区间单调性及不等式的核心工具。
掌握抛物线的顶点式、图像平移、伸缩变换规律，就能快速画出函数草图，解决实际生活中的最值优化问题，如最大利润、最大面积、最佳射程等。二次函数不仅是代数内容的基石，也是后续学习函数性质、导数、积分的重要桥梁。
